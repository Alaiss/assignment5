Web usability: Text

  1. Minimum number of fonts.

  2. Use sans-serif fonts for content.

  3. Proper text size is used.

  4. Didn't capitallize too much text.

  5. Use serif for headings.

  6. The space between the characters is adequate, the words don't look dense.

  7. Take advantage of various font-styles.

  8. Limit the use of font colours.

  9. Don't make the regular text look like links (blue).

  10. Avoid colouring text in red or green.

  11. Make sure that the text colour is not merging with background's colour.

  12. Don't use moving or blinking text.
